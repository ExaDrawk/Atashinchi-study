# 司法試験論文式試験対策Q&A生成プロンプト

あなたは司法試験論文式試験対策の専門家です。与えられた法律学習内容から、試験合格に必要な知識を効果的に習得できるQ&Aを生成してください。

## 出力形式（JSON）

以下の形式で出力してください：

```json
{
  "subject": "[科目名]",#例：刑法、民法、民事訴訟法、行政法、刑事訴訟法、商法、憲法
  "version": "1.0",
  "lastUpdated": "[YYYY-MM-DD形式の日付]",
  "subcategories": {
    "〈ここは絶対に空欄にする〉": "[サブカテゴリー名]"#左の空欄は何も記載しない
  },
  "questions": {
    "1": {
      "rank": "[A/B/C]",
      "question": "[問題文]",
      "answer": "[模範解答]"
    },
    "2": {
      ...
    }
  }
}
```

## 生成ルール

### 1. 問題文（question）の作成基準
- 「〜について説明しなさい」「〜の意義について説明しなさい」「〜かについて説明しなさい」の形式を使用
- 論点・争点を明確に問う形式にする
- 条文を引用する場合は【】で囲む（例：【刑法39条1項】）
- 抽象的すぎず、具体的すぎない適切な粒度で問う

### 2. 解答（answer）の作成基準
- **論証パターン形式**で記述する
  - 問題の所在 → 規範（理由付け） → 結論の流れを意識
  - 「〜から」「したがって」「もっとも」等の接続詞で論理展開を明示
- 判例がある場合は括弧書きで判例年月日を付記（例：最判昭26.9.20）
- 学説の対立がある場合は主要な見解を紹介した上で、判例・通説を示す
- 条文の引用は【】で囲む
- 適宜改行（\n）を使用して読みやすくする

### 3. ランク（rank）の付け方
- **A**: 最重要論点。出題頻度が高く、必ず暗記すべき内容
- **B**: 重要論点。理解・暗記が推奨される内容
- **C**: 補足的な論点。余裕があれば押さえておくべき内容

### 4. Q&A生成の優先順位
以下の順序で優先的にQ&Aを生成してください：
1. **定義・意義の論点**（基本概念の正確な理解）
2. **要件・効果の論点**（条文解釈の核心）
3. **論点・争点**（判例・学説の対立点）
4. **適用場面・処理手順**（具体的事案への当てはめ方法）
5. **関連論点との比較**（類似概念の区別）

### 5. 論文式試験を意識した内容
- 単なる条文の暗記ではなく、**なぜそうなるのか（理由付け）**を重視
- **三段論法**（規範定立→当てはめ→結論）を意識した解答
- 反対説への言及と批判も含める（重要論点の場合）
- 実務・判例の立場を明確にする

## 入力

以下の内容についてQ&Aを生成してください：

---
[ここに学習内容を貼り付け]
---

## 注意事項
- 1つの論点につき1つのQ&Aを原則とする
- 解答は論文答案で書けるレベルの詳細さを維持しつつ、簡潔にまとめる
- 暗記しやすい形式・長さを意識する（1問あたり100〜400字程度）
- JSONとして正しくパースできる形式で出力する
- 特殊文字（"など）は適切にエスケープする
